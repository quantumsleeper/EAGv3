# 🧠 Semantic History Replay

A powerful Chrome Extension and Python backend that transforms your browser history into a deeply searchable, multimodal vector database. Powered by **Gemini Flash** and **ChromaDB**.

Never lose a webpage again. Instead of furiously searching your Chrome history for exact URLs or title text, simply describe what you saw:
> *"That website with the dark green header... "*
> *"The blog post containing a chart about transformer scaling laws..."*

## ✨ Features

- **Passive Multimodal Capture**: Invisibly takes a low-res snapshot and extracts text content if you stay on a webpage for 5 seconds.
- **Privacy-First Blacklist**: Automatically ignores sensitive URLs (passwords, banking, admin panels, auth flows).
- **Deep Semantic Analysis**: Uses Google's generative AI to deeply analyze the visual layout and text of every page you visit, translating it into a highly detailed indexable description.
- **Instant Vector Search**: Searches against a local `ChromaDB` database entirely on your own machine.
- **Modern Search UI**: A beautiful, glassmorphic popup interface complete with micro-animations and screenshot previews.

---

## 🏗️ Architecture

This project is separated into a two-part architecture:

1. **Frontend (Chrome Extension / Manifest V3)**
   - Resides in the `/extension` directory.
   - A Service Worker (`background.js`) tracks page visits, applies the sensitive-URL blacklist, waits 5 seconds, captures the visible tab, and sends the payload to the local server.
   - The UI (`popup.html` & `styles.css`) provides the search box and renders the fetched search results from the backend.

2. **Backend (Python FastAPI)**
   - Resides in the `/backend` directory.
   - Exposes a `/ingest` REST endpoint to receive base64 screenshots and page text.
   - Processes payloads via **Gemini Flash** through the `google-generativeai` SDK to build complete visual & semantic context.
   - Embeds and stores the descriptions inside a local isolated vector database using `chromadb`.
   - Exposes a `/search` REST endpoint for natural language history retrieval.

## 🚀 Getting Started

### Prerequisites
- Python 3.11+
- Google Gemini API Key

### 1. Backend Setup

1. Open your terminal and change directories to the backend folder:
   ```bash
   cd backend
   ```
2. Create and activate a Python virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Configure your API key:
   - Copy the `.env.example` file and rename it to `.env`.
   - Insert your actual Gemini API key inside:
     ```env
     GEMINI_API_KEY=your_actual_api_key_here
     ```
5. Start the FastAPI server:
   ```bash
   ./venv/bin/python -m uvicorn main:app --reload --port 8000
   ```
   > **Note (macOS Users)**: To avoid overriding the virtual environment namespace when multiprocessing (since Uvicorn spawns child processes for `--reload`), be sure to spawn Uvicorn strictly through module execution (`python -m uvicorn`) using your local virtual environment binary as showcased above.

### 2. Chrome Extension Setup

1. Open Google Chrome and navigate to: `chrome://extensions/`
2. Toggle on **Developer Mode** in the top right corner.
3. Click **Load Unpacked**.
4. Select the `extension/` folder inside this repository.
5. Pin the Semantic History Replay extension to your Chrome toolbar for easy access.

## 🧪 Testing it Out

1. **Populate the Database:** 
   Browse the web! Visit 3 or 4 visually distinct websites (e.g. Wikipedia, a Github repo, a colorful landing page). Stay on each page for at least 5 seconds so the background script captures it.
2. **Verify Logs:**
   Look at your Python terminal running the Uvicorn server. You should see printouts confirming that `POST /ingest` was received and indexed.
3. **Run a Semantic Search:** 
   Click the extension icon in your Chrome toolbar. Try searching for visual cues or abstract themes related to the sites you visited (e.g., *"dark theme dashboard"* or *"that article about open-source AI"*). You will instantly receive a card displaying a screenshot of your visit with a direct link.

> **Privacy Tip**: If you want to wipe your local semantic history, simply stop your backend server and delete the autogenerated `/screenshots` and `/chroma_data` directories inside the `backend` folder.

## 🛠️ Tech Stack

- **Extension**: Vanilla JS, Manifest V3, HTML/CSS
- **Server**: FastAPI, Uvicorn, Python 3.11
- **Vector DB**: ChromaDB, SQLite3
- **AI Models**: Google Gemini (`gemini-flash-latest`), `all-MiniLM-L6-v2` (for ChromaDB Text Embedding)
